# Setting up the Polygon ID Issuer Node

This walks through standing up the [Privado ID / Polygon ID issuer node](https://github.com/0xPolygonID/issuer-node) locally, as run in this POC (reconstructed from the recorded terminal session in `Terminal Saved Output.txt`).

## 1. Clone the issuer node

```bash
git clone https://github.com/0xPolygonID/issuer-node.git
cd issuer-node
```

## 2. Start the infrastructure containers

Spin up Redis, Postgres, and Vault (used for private key storage):

```bash
make up
```

This runs `docker compose -f infrastructure/local/docker-compose-infra.yml up -d redis postgres vault`.

## 3. Import the issuer's Ethereum private key into Vault

```bash
make private_key=<your_private_key> add-private-key
```

This writes the key into Vault at `iden3/import/pbkey` (`key_type=ethereum`).

## 4. Clean up the Vault token env var

```bash
make add-vault-token
```

## 5. Generate the issuer DID

```bash
make generate-issuer-did-arm   # or `make generate-issuer-did` on non-ARM
```

This starts the `initializer` container, runs the DB migrations, and creates the identity. On success you'll see a `did:polygonid:polygon:mumbai:...` identifier printed to the logs — this is your issuer's DID on the Polygon Mumbai testnet.

## 6. Run the issuer UI and API

```bash
make run-ui-arm   # or `make run-ui`
```

This brings up `api-ui`, `ui`, `notifications`, and `pending_publisher` containers. The issuer UI is then reachable locally (default `http://localhost`), where you can define credential schemas and issue credentials to a DID.

## 7. Expose the issuer node publicly with ngrok

The Polygon ID mobile wallet needs to reach your issuer node over the internet to scan QR codes and pull credentials/proof requests. Since the issuer node runs locally, use ngrok to tunnel it:

```bash
ngrok http 80
```

Update the issuer node's public URL configuration (`.env-issuer` / `.env-api`, per the issuer node docs) to the ngrok HTTPS URL so QR codes generated by the UI point to a publicly reachable endpoint. See `docs/ngrok-notes.md` for the specific gotchas hit during this POC.

## Tearing down

```bash
make down          # stop containers
make clean-vault    # wipe local Vault state (data/init.out, core, logical, sys)
```
